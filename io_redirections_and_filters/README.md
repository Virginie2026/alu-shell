# Shell, I/O Redirections and filters

This project contains scripts for shell I/O redirections and filters.

## Scripts

* 0-hello_world: prints Hello, World
* 1-confused_smiley: displays a confused smiley face
* 7-file: creates a file
* 8-cwd_state: writes the result of ls -la to a file
* 9-duplicate_last_line: duplicates the last line of a file
* 10-no_more_js: deletes all .js files
* 11-directories: counts directories and sub-directories
* 12-newest_files: displays the 10 newest files
* 13-unique: prints words that appear exactly once
* 14-findthatword: displays lines containing the pattern
* 15-countthatword: displays the number of lines containing pattern
* 16-whatsnext: displays lines containing pattern and 3 lines after
* 17-hidethisword: displays lines not containing pattern
* 18-letteronly: displays lines with letters only
* 19-AZ: replaces characters
* 20-hiago: removes letters
* 21-reverse: reverses input
* 22-users_and_homes: displays users and home directories
* 23-empty_casks: finds empty files and directories
* 24-gifs: lists gif files
* 25-acrostic: decodes acrostic
* 26-the_biggest_fan: parses web servers logs
